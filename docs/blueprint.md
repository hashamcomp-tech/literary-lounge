# **App Name**: Literary Lounge

## Core Features:

- Novel Display: Display novels in an easy to read format.
- Chapter Navigation: Navigate between chapters within a novel.
- Search Functionality: Allow users to search for specific novels or chapters.
- Personalized Recommendations: Suggest novels based on reading history and preferences using a recommendation tool.
- Reading Progress: Track the user's reading progress and remember the last read position.

## Style Guidelines:

- Primary color: Soft Indigo (#7986CB) for a calming reading environment.
- Background color: Light gray (#F0F2F5) to ensure readability.
- Accent color: Muted Violet (#9575CD) for interactive elements and highlights.
- Body font: 'Literata', a serif font known for its readability in long texts.
- Headline font: 'Literata', consistent use ensures a literary feel.
- Simple, elegant icons for navigation and settings.
- Subtle animations for page transitions and interactive elements to enhance user experience.
